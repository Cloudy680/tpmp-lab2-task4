## Описание
Программа для обработки данных об автомобилях:
- Структура "Автомобиль": марка, цвет, серийный номер, регистрационный номер, год выпуска, год техосмотра, цена
- Чтение данных из текстового файла
- Поиск автомобилей марки Audi, возраст которых больше 3 лет
- Запись результатов в новый файл

## Модель КИС
- `include/car.h` — интерфейс (определение структуры и функций)
- `src/car.c` — сервер (реализация функций)
